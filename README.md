# BOIBOI Course Support

A simple web-based course management application that allows users to add, edit, delete, and view their courses. Built with HTML, CSS, and JavaScript, featuring local storage for data persistence.

## Features

- **Add Courses**: Easily add new courses with title, code, and units.
- **Edit Courses**: Modify existing course details.
- **Delete Courses**: Remove courses with confirmation.
- **View Courses**: Display all courses in a clean table format.
- **Persistent Storage**: Uses browser localStorage to save data across sessions.
- **Responsive Design**: Modern UI with a dark theme and smooth animations.
- **Inspirational Quotes**: Random educational quotes displayed on the side.

## Technologies Used

- **HTML5**: Structure and layout
- **CSS3**: Styling and animations
- **JavaScript (ES6+)**: Functionality and DOM manipulation
- **Local Storage API**: Data persistence

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/boiboi-course-support.git
   cd boiboi-course-support
   ```

2. Open `index.html` in your web browser, or serve it with a local server:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000`
